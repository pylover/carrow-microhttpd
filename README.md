# carrow-microhttpd

An integration with Carrow and Microhttpd.

This webserver will echo back whatever you pass as a url.


## Build

### Dependencies

- [Microhttpd](./README)


## Clone
```bash
git clone git@github.com:pylover/carrow-microhttpd.git
cd carrow-microhttpd
```

## Lint
```bash
make lint
```
